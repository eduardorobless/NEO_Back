NEO (Near Earth Object) project

Hazardous Near Earth Objects, look up to the sky before you go out!

Aim of project

The aim of this project is apply backend technologies to get relevant data for the human species; the Near Earth Objects!

How it works?

How it works? Well, the app works in the following way:
  • We use the NASA API Asteroids - NeoWs. Being a well respected organization the NASA provides several APIs to gather data and make it useful to people.
  • Apply start date and end date parameter to delimitate gather date ranges (we use the same date for both).
  • Get the data back in form of JSON object

The technologies used are

  • Express. To create the RESTful API that can call the NASA API
  • Postman. To test the RESTful API

API details

  • Method: Get
  • Endpoint: localhost:5000/neo?start_date={yyyy-mm-dd}&end_date={yyyy-mm-dd}
